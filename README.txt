Note: base directory is projects/murders/

We analyze US gun murder total collected by FBI
01-download-data.R - downloads murders.csv data in the data directory
02-wrangle-data.R - creates murders.rda dataset and saves as R data.file in rda directory
03-data-analysis.r - generates barplot-by-state.png plot and saves it in figs directory
04-report.Rmd - generates the analysis report in .html format
